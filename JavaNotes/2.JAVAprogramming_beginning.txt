1. 프로그래밍이란
    1)프로그래밍이란(programming)    
        -컴퓨터가 일을 수행하도록 프로그래밍 언어로 명령어 집합을 만드는 일
    2)컴파일(compile)
        -프로그래밍 언어를 컴퓨터가 실행 가능한 기계어로 만드는 일
    3)Compiler
        -기계어로 바꾸어 주는 프로그램 (javac.exe, gcc(linux))

3.자바프로그램의 특징
    1)용어정리
        -JDK Java Development Kit 개발용 라이브러리 JRE+JVM+a 
        -JRE Java Runtime Environment 실행 환경
        -JVM Java virtual machine 자바 프로그램이 실행되는 환경 JRE에 포함되어있음
    2)플랫폼에 영향을 받지 않아 다양한 환경에서 사용가능
        -Write Once, Run Anywhere  jvm덕에 밑 단의 신경을 안써도 되게 됨
    3)객체지향언어=유지보수 쉽고 확장성 좋음
    4) 프로그램이 안정적
    5) 풍부한 기능이 제공되는 오픈소스
